Maintenance Mode And Under Construction Page plugin with Launch page & site offline plugin for your Website while it's under construction.

== Description ==

If your site is under development then use this plugin to display an under maintenance / under construction page to your visitors. The plugin allows collecting all visitors subscriptions. When your site will live plugin notify all your active subscriber about site launch.

== Installation ==

1. Upload the entire `maintenance-mode-and-under-construction-page` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Weblizar Coming Soon menu page to customize plugin settings.
4. Live it & Enjoy.
